## How to use

```bash
# install hexo-cli
npm install -g hexo-cli

# create a new blog
hexo new "My New Blog"

# generate static files
yarn generate

# deploy to github pages
yarn deploy

# or you can use the following command to deploy (it can handle the cname file)
yarn go
```
